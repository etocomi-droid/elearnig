6.1　メール配信でできること

メール配信機能は、お客さまにメールを送る仕組みです。2種類の送り方があります。

種類たとえると…使い道
シナリオ（ステップメール）「入学おめでとう→1週間後に宿題→2週間後にテスト」のように決まった順番で届く手紙申し込んでくれた人に、日にちをあけて順番に情報を届ける
一斉配信（キャンペーン）全員に同じチラシを配るお知らせやセールの案内を全員に一度に送る

左メニューの 「メール配信」 を押すと、メール配信のダッシュボード画面が表示されます。ここから「シナリオ」タブと「一斉配信」タブを切り替えて操作します。

6.2　シナリオ（ステップメール）を作る

シナリオとは、「申し込んだ日から数えて、○日後にこのメールを送る」という設定です。一度作っておけば、新しいお客さまが申し込むたびに自動でメールが届きます。

シナリオを作る手順

  1. 左メニューの 「メール配信」 を押します
  2. 「シナリオ」 タブが選ばれていることを確認します
  3. 右上の 「新規シナリオ作成」 ボタンを押します
  4. シナリオ名を入力します（例: 無料セミナーフォローアップ）
  5. 「作成」 ボタンを押します

ステップ設定の例

ステップ1: 0日後 → 「お申込みありがとうございます」 ステップ2: 1日後 → 「明日のセミナーについてのご案内」 ステップ3: 3日後 → 「セミナーはいかがでしたか？」 ステップ4: 7日後 → 「特別なご案内があります」
ポイント: シナリオをファネルのフォームと連携させると、フォームに入力があった時点で自動的にメール配信が始まります（第5章「フォームブロックの設定」参照）。

パーソナライズ（お客さまの名前を自動挿入する）

メールの本文の中に特別なキーワードを入れると、お客さまごとに名前やメールアドレスを自動で差し替えて送ることができます。これを「パーソナライズ」と呼びます。

キーワード置き換わる内容使用例
{{ contact.last_name }}お客さまの姓「{{ contact.last_name }}様、こんにちは。」→「田中様、こんにちは。」
{{ contact.first_name }}お客さまの名「{{ contact.first_name }}さん」→「太郎さん」
{{ contact.email }}お客さまのメールアドレス確認メールなどに活用
ヒント: パーソナライズを使うと、一通一通手書きしたようなメールが自動で届きます。「○○様」と名前入りのメールは、お客さまに特別感を感じてもらえるのでおすすめです。
注意: キーワードは {{ と }} で囲む書き方を正確に守ってください。スペルを間違えると、キーワードがそのまま表示されてしまいます。

シナリオの活用ヒント

  • ウェルカムメール: 申し込み直後（0日後）に、お礼と今後の流れを伝えるメールを送りましょう
  • リマインダーメール: セミナーやイベントの前日にリマインドメールを送ると、参加率が上がります
  • フォローアップメール: 数日後に「いかがでしたか？」と聞くメールを送ると、お客さまとの関係が深まります
  • セールスメール: 信頼関係ができたタイミング（1〜2週間後）で、有料商品のご案内を送るのが効果的です

6.3　一斉配信（キャンペーン）を作る

全員に同じメールを一度に送りたいときは、一斉配信を使います。

手順

  1. メール配信画面で 「一斉配信」 タブを押します
  2. 右上の 「新規一斉配信作成」 ボタンを押します
  3. 以下を入力します
入力欄何を入れるか例
配信名この配信の管理名夏のセールお知らせ
件名メールのタイトル【期間限定】夏のセールのお知らせ
本文（テキスト）メールの文面いつもありがとうございます…
本文（HTML）装飾付きの文面（任意）HTMLがわからなければ空でOK
対象タグ特定のお客さまだけに送りたい場合に選択購入済み
  1. 「保存」 ボタンを押します

ヒント: 一斉配信でもパーソナライズが使えます。本文に {{ contact.last_name }} と入力すると、お客さまの名前が自動的に挿入されます。全員に同じ内容を送る場合でも、名前入りにするだけで開封率が上がります。
ヒント: 「対象タグ」を使うと、特定のお客さまだけにメールを送れます。たとえば「購入済み」タグがついたお客さまだけに新商品の案内を送る、といった使い方ができます。タグを選ばなければ、全員に送信されます。

メールを送信する

  1. 保存した一斉配信の編集画面を開きます
  2. 内容を確認します
  3. 「送信」 ボタンを押します
  4. 確認メッセージが出たら 「OK」 を押します
注意: 送信したメールは取り消せません。送信前に必ず内容を確認しましょう。特に件名の誤字脱字や、リンク先のURLが正しいかを確認してください。

メール配信のコツ

  • 件名は短く、具体的に: 長すぎる件名はスマートフォンで途中で切れてしまいます。20〜30文字を目安にしましょう
  • 送信時間を考える: 一般的に、平日の午前10時〜12時、または夜の20時〜21時ごろが開封されやすいと言われています
  • 本文は読みやすく: 1つの段落は3〜4行程度にして、適度に改行を入れると読みやすくなります
  • 行動を促す: メールの最後には「詳しくはこちら」「お申し込みはこちらから」のようにリンクを入れましょう

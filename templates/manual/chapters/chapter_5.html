5.1　ファネルとは

ファネルとは、インターネット上に公開するページ（ホームページのようなもの）のことです。

たとえば、次のようなページを簡単に作れます。

  • 無料セミナーの申し込みページ
  • 商品の紹介ページ
  • お礼のページ（「お申し込みありがとうございます」と表示するページ）

このシステムでは、プログラミングの知識がなくても、ブロック（部品）を並べるだけでページが作れます。積み木を組み合わせるイメージです。

ポイント: ファネルは「漏斗（ろうと）」という意味で、マーケティングの世界でよく使われる言葉です。たくさんの人にページを見てもらい、その中から興味を持った方が申し込んでくれる、という流れをイメージしています。難しく考えず、「自分のウェブページを作る機能」と覚えてください。

5.2　新しいファネルを作る

手順

  1. 左メニューの 「ファネル」 を押します

すると、ファネルの一覧画面が表示されます。すでに作成したファネルがあれば、ここに一覧で表示されます。

ファネル一覧画面
左メニューの「ファネル」を押すと表示される画面です。作成済みのファネルが一覧で表示され、右上に「新規ファネル作成」ボタンがあります。
  1. 右上の 「新規ファネル作成」 ボタンを押します
  2. 以下を入力します
入力欄何を入れるか例
ファネル名このページセットの名前（管理用なのでお客さまには見えません）無料セミナー申込
スラッグ短い英語の名前（URLの一部になります）free-seminar
ファネル作成フォーム
「新規ファネル作成」ボタンを押すと表示される入力フォームです。ファネル名とスラッグを入力する欄があり、下部に「作成」ボタンがあります。
  1. 「作成」 ボタンを押します
スラッグとは？: スラッグは、ページのURL（インターネットの住所）の一部になる短い英語の文字列です。たとえばスラッグを「free-seminar」にすると、ページのURLは /p/free-seminar/... のようになります。半角の英数字とハイフン（-）だけ使えます。

5.3　ページを追加する

1つのファネルの中に、複数のページを作れます。たとえば「申し込みページ」と「ありがとうページ」を1つのファネルにまとめることができます。

手順

  1. 作成したファネルの編集画面で、「ページ追加」 ボタンを押します
  2. 以下を入力します
入力欄何を入れるか例
ページタイトルページの名前セミナー紹介ページ
スラッグ短い英語の名前（URLの一部になります）intro
ページの種類用途を選ぶランディングページ
  1. 「作成」 ボタンを押します
ページ一覧画面
ファネル編集画面内のページ一覧です。追加済みのページがリストで表示され、各ページの名前・スラッグ・種類が確認できます。右上に「ページ追加」ボタンがあります。

ページの種類について

種類使い道
ランディングページ最初にお客さまが見るページ（チラシのようなもの）
サンクスページ申し込み後に表示する「ありがとう」ページ
セールスページ商品を詳しく紹介するページ
フォームページお名前やメールアドレスを入力してもらうページ
アップセルページ追加の商品をおすすめするページ
ヒント: 最初は「ランディングページ」と「サンクスページ」の2つだけ作れば十分です。ランディングページで商品やサービスの魅力を伝え、フォームで申し込みを受け付け、サンクスページで「ありがとうございます」と表示する、というのが基本の流れです。

5.4　ページにブロック（セクション）を追加する

ページの中身は「セクション」というブロック（部品）を積み重ねて作ります。

手順

  1. ページ編集画面の右側にある 「セクション追加」 ボタンを押します
  2. 追加したいブロックの種類を選びます
  3. ブロックが追加されます
  4. ブロックの編集ボタン（鉛筆マーク）を押して、内容を書き換えます
セクション追加画面
ページ編集画面で「セクション追加」ボタンを押すと表示されるセクション種類の選択画面です。ヒーロー、テキスト、画像、動画、CTAボタン、フォームなどのブロックが一覧で並んでいます。

ブロック（セクション）の種類

種類できることたとえると…
ヒーロー大きな見出しと説明文看板・ポスターの見出し
テキスト自由な文章チラシの本文
画像写真やイラストの表示チラシに貼る写真
動画YouTube等の動画を埋め込むテレビCM
CTAボタン押せるボタン（申込ボタンなど）「ここを押してください」ボタン
フォーム名前やメールの入力欄申込用紙
カウントダウン締め切りまでの時間表示残り時間タイマー
お客様の声体験者のコメント表示口コミ・レビュー
FAQよくある質問と答えQ&Aコーナー
区切り線ブロック間の仕切りページの区切り

各ブロックの編集方法（詳細）

ヒーローブロック

ヒーローブロックは、ページの一番上に置く「目を引く見出し」のブロックです。お客さまがページを開いたとき、最初に目に入る部分なのでとても大切です。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 見出し（大きな文字）を入力します（例: 「たった3日で料理の腕が上がる！」）
  3. サブテキスト（見出しの下の説明文）を入力します（例: 「初心者でも安心のオンライン講座」）
  4. 必要に応じて背景色を変更します
  5. 「保存」 を押します
ヒント: ヒーローの見出しは短く、お客さまの心に響く言葉にしましょう。「何が得られるか」を伝えるのが効果的です。

テキストブロック

テキストブロックは、自由に文章を書けるブロックです。商品やサービスの説明、あいさつ文など、好きな内容を書けます。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 本文のテキストエリアに文章を入力します
  3. 必要に応じて太字や色を変えて、大切な部分を目立たせましょう
  4. 「保存」 を押します

画像ブロック

画像ブロックは、写真やイラストをページに表示するブロックです。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 画像URLにインターネット上の画像のアドレスを入力するか、画像をアップロードします
  3. 必要に応じて代替テキスト（Alt）を入力します（画像が表示できないときに代わりに表示される文字です）
  4. 「保存」 を押します
ヒント: 画像は商品やサービスの魅力が伝わるものを選びましょう。人物の写真や、実際の作業風景の写真は効果的です。

動画ブロック

動画ブロックは、YouTubeなどの動画をページに埋め込むブロックです。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 動画のURLを入力します（YouTubeの動画ページのURLをそのまま貼り付けます）
  3. 「保存」 を押します
ヒント: YouTubeの動画URLは、ブラウザのアドレスバーからコピーして貼り付けるだけでOKです。例: https://www.youtube.com/watch?v=XXXXX

CTAボタンブロック

CTAボタンブロックは、お客さまに押してほしい「行動を促すボタン」を作るブロックです。CTA は「Call To Action（行動の呼びかけ）」の略です。

  1. 鉛筆マーク（編集ボタン）を押します
  2. ボタンのテキストを入力します（例: 「今すぐ申し込む」「詳しくはこちら」）
  3. リンク先URLを入力します（ボタンを押したときに移動するページのアドレス）
  4. 必要に応じてボタンの色を変更します
  5. 「保存」 を押します
ヒント: ボタンの文字は「今すぐ申し込む」「無料で参加する」のように、具体的な行動を書くと効果的です。「こちら」だけでは何のボタンかわかりにくくなります。

フォームブロック

フォームブロックは、お客さまに名前やメールアドレスを入力してもらうためのブロックです。詳しい設定は次のセクション（5.5）で説明します。

カウントダウンブロック

カウントダウンブロックは、締め切りまでの残り時間を表示するブロックです。「あと3日！」のように表示されるので、お客さまに「早く申し込まなきゃ」と感じてもらえます。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 終了日時を設定します（カウントダウンのゴール）
  3. 必要に応じて表示スタイルを変更します
  4. 「保存」 を押します

お客様の声ブロック

お客様の声ブロックは、実際にサービスを使ったお客さまの感想（レビュー）を表示するブロックです。

  1. 鉛筆マーク（編集ボタン）を押します
  2. お客さまの名前を入力します（例: 「田中さま」「A.T.さん」）
  3. コメント内容を入力します（例: 「とてもわかりやすかったです！」）
  4. 必要に応じて複数の声を追加します
  5. 「保存」 を押します
ヒント: お客様の声は、サービスの信頼感を高めるためにとても効果的です。できるだけ具体的なコメントを掲載しましょう。

FAQブロック

FAQブロックは、「よくある質問とその答え」を表示するブロックです。お客さまの不安を解消するのに役立ちます。

  1. 鉛筆マーク（編集ボタン）を押します
  2. 質問を入力します（例: 「初心者でも大丈夫ですか？」）
  3. 回答を入力します（例: 「はい、初心者の方でも安心して受講いただけます。」）
  4. 必要に応じて質問と回答のペアを追加します
  5. 「保存」 を押します

区切り線ブロック

区切り線ブロックは、ブロックとブロックの間に横線を引いて見た目を整えるブロックです。特に編集項目はありません。追加するだけで使えます。

セクション編集画面
ブロックの鉛筆マーク（編集ボタン）を押すと表示される編集画面です。各ブロックの種類に応じて、見出し・テキスト・画像URLなどの入力欄が表示されます。下部に「保存」ボタンがあります。

ブロックの並び替え

ブロックの順番を変えたいときは、ブロックをマウスで掴んで上下にドラッグ（引きずる）します。好きな順番に並べ替えられます。

ブロックの編集

  1. 変更したいブロックの 鉛筆マーク（編集ボタン）を押します
  2. 内容を書き換えます
  3. 「保存」 を押します

ブロックの削除

  1. 削除したいブロックの ゴミ箱マーク（削除ボタン）を押します
  2. 確認メッセージが出たら 「OK」 を押します
注意: ブロックを削除すると、そのブロックに入力した内容はすべて消えます。元に戻せませんので、削除する前に内容を確認しましょう。

5.5　フォームブロックの設定（重要）

フォームブロックは、お客さまに名前やメールアドレスを入力してもらうためのブロックです。これがあることで、お客さまの情報が自動的にコンタクト（顧客台帳）に登録されます。

フォームの設定手順

  1. フォームブロックの 鉛筆マーク（編集ボタン）を押します
  2. 以下の項目を設定します

フォームの設定項目

項目意味例
送信ボタンの文字ボタンに表示する文字無料で申し込む
シナリオID申し込み後に自動メールを送る設定（後述）1
リダイレクトURL申し込み後に表示するページのアドレス/p/free-seminar/thanks/
  1. 設定が終わったら 「保存」 を押します
フォームブロックの設定画面
フォームブロックの編集画面です。送信ボタンの文字、シナリオID、リダイレクトURLなどの入力欄が表示されます。シナリオIDを設定すると、フォーム送信時に自動でメール配信が開始されます。
ポイント: シナリオIDは、第6章「メール配信」で作成するシナリオの番号です。フォームとシナリオを連携させると、お客さまがフォームに入力した瞬間に自動でメールが届くようになります。とても便利な機能なので、ぜひ活用しましょう。
注意: リダイレクトURLを設定しないと、フォーム送信後にお客さまが同じページにとどまってしまいます。必ず「ありがとうページ（サンクスページ）」のURLを設定しましょう。

5.6　ページを公開する

作ったページをお客さまが見られるようにするには、「公開」の設定をします。

手順

  1. ファネルの編集画面で 「公開する」 にチェックを入れます
  2. 「更新」 ボタンを押します
  3. 公開URLが表示されます（例: /p/free-seminar/intro/）
  4. このURLにアクセスすると、作ったページが表示されます
公開されたページのプレビュー
ページを公開した後、公開URLにアクセスしたときの表示です。設定したヒーロー、テキスト、画像、フォームなどのブロックが順番に表示され、お客さまが閲覧・操作できる状態になっています。
ポイント: 公開をやめたい場合は、チェックを外して「更新」を押せば、ページは非公開になります。非公開にすると、URLにアクセスしてもページは表示されません。
ヒント: 公開する前に、一度自分でURLにアクセスして、ページの見た目や動作を確認しましょう。スマートフォンからも確認すると、お客さまがどのように見るかがわかります。

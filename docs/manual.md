# 第1章　このシステムでできること

## 1.1　システムの全体像

このシステムは、**インターネットを使ってお客さまを集め、商品を販売し、学習コンテンツを届ける**ための道具箱です。

たとえるなら、次のようなことが一つの場所でまとめてできます。

| やりたいこと | たとえると… | このシステムでは |
|---|---|---|
| お客さまの情報を管理したい | 名刺ファイル・顧客台帳 | **コンタクト管理** |
| 申し込みページを作りたい | チラシ・ポスター | **ファネル（集客ページ）** |
| メールを自動で送りたい | ダイレクトメール | **メール配信** |
| 商品を売って代金をもらいたい | レジ・請求書 | **商品・決済** |
| 動画や教材を届けたい | 教室・スクール | **会員サイト** |

## 1.2　使うまでの大まかな流れ

```
① アカウントを作る（ユーザー登録）
    ↓
② プロジェクトを作る（お店の看板をかける）
    ↓
③ 集客ページを作る（チラシを作る）
    ↓
④ メール配信を設定する（お知らせの手紙を準備する）
    ↓
⑤ 商品を登録する（売り物を棚に並べる）
    ↓
⑥ 会員サイトを作る（教室を開く）
    ↓
⑦ お客さまが来る → 自動でメールが届く → 商品を買う → 教材が見られる
```

**ポイント**: すべてブラウザ（インターネットを見る画面）だけで操作できます。特別なソフトをインストールする必要はありません。

# 第2章　はじめかた（アカウント作成とログイン）

## 2.1　アカウントを作る（ユーザー登録）

アカウントとは、このシステムにあなた専用の「入り口」を作ることです。メールアドレスと合言葉（パスワード）を決めるだけでできます。

### 手順

1. ブラウザで **http://127.0.0.1:8000/signup/** を開きます
2. 以下の4つを入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| ユーザー名 | 好きな名前（半角英数） | `tanaka` |
| メールアドレス | あなたのメールアドレス | `tanaka@example.com` |
| パスワード | 合言葉（8文字以上） | `MyPass1234` |
| パスワード（確認） | 上と同じものをもう一度 | `MyPass1234` |

3. **「アカウントを作成」** ボタンを押します
4. 成功すると、自動的に次の画面に移ります

> **注意**: パスワードは忘れないようにメモしておきましょう。

## 2.2　ログインする

すでにアカウントがある場合は、ログイン画面から入ります。

### 手順

1. ブラウザで **http://127.0.0.1:8000/login/** を開きます
2. ユーザー名とパスワードを入力します
3. **「ログイン」** ボタンを押します
4. 成功すると、トップ画面（ダッシュボード）が表示されます

## 2.3　プロジェクトを作る

プロジェクトとは、「お店」や「事業」のようなものです。たとえば「料理教室」や「英会話スクール」など、目的ごとにプロジェクトを分けて管理できます。

### 手順

1. 初めてログインすると、プロジェクト作成画面が表示されます
2. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| プロジェクト名 | 事業やお店の名前 | `田中料理教室` |
| スラッグ | 短い英語の名前（自動入力も可） | `tanaka-cooking` |

3. **「作成」** ボタンを押します
4. ダッシュボード画面に移ります

> **「スラッグ」とは？**: インターネットの住所（URL）の一部になる短い名前です。半角英数字とハイフン（-）だけが使えます。

## 2.4　プロジェクトを切り替える

複数のプロジェクトがある場合、画面左上のプロジェクト名を押すと、別のプロジェクトに切り替えられます。

### 手順

1. 画面左側の一番上にあるプロジェクト名を確認します
2. プロジェクト一覧画面（ `/projects/` ）で切り替えたいプロジェクトの **「切替」** ボタンを押します
3. 画面の内容が、選んだプロジェクトのものに切り替わります

## 2.5　ログアウトする

画面右上のあなたのユーザー名の隣にある **「ログアウト」** ボタンを押すと、安全にシステムから退出できます。

# 第3章　ダッシュボード（トップ画面）の見かた

## 3.1　ダッシュボードとは

ダッシュボードは、ログインすると最初に表示される**「本部」**のような画面です。あなたのプロジェクトの状況がひと目でわかります。

## 3.2　画面の説明

### 左側：メニュー（ナビゲーション）

画面の左側にある黒い帯がメニューです。ここから各機能に移動できます。

| メニュー名 | できること |
|---|---|
| ダッシュボード | この画面に戻る |
| コンタクト | お客さまリストを見る |
| ファネル | 集客ページを管理する |
| メール配信 | メールの設定をする |
| 商品管理 | 売り物を管理する |
| 会員サイト | 学習コーナーを管理する |

### 中央上部：数字のカード（4つ）

| カードの色 | 表示内容 | 意味 |
|---|---|---|
| 青 | コンタクト数 | 登録されているお客さまの人数 |
| 紫 | ファネル数 | 作った集客ページの数 |
| 緑 | 注文数 | 受けた注文の数 |
| オレンジ | シナリオ数 | 作ったメール自動配信の数 |

### 中央下部：かんたん操作ボタン（3つ）

よく使う操作にすぐ移動できるボタンです。

- **顧客を管理する** → コンタクト一覧へ
- **新しいファネルを作る** → ファネル作成へ
- **シナリオを管理する** → メール配信設定へ

# 第4章　コンタクト管理（お客さまリスト）

## 4.1　コンタクトとは

コンタクトとは、**あなたのサービスに興味を持ってくれたお客さま**のことです。名前やメールアドレスを登録しておくことで、メールを送ったり、購入履歴を見たりできます。

紙の「顧客台帳」のデジタル版だと思ってください。

## 4.2　コンタクト一覧を見る

### 手順

1. 左メニューの **「コンタクト」** を押します
2. 登録されているお客さまの一覧が表示されます

### 画面の見かた

一覧には以下の情報が表示されます。

| 列の名前 | 内容 |
|---|---|
| 名前 | お客さまの名前（押すと詳しい情報が見られます） |
| メールアドレス | お客さまの連絡先 |
| タグ | お客さまにつけた目印（色付きのラベル） |
| 登録日 | いつ登録されたか |
| 操作 | 「詳細」ボタン |

## 4.3　お客さまを検索する

お客さまが増えてきたら、検索機能を使って探すことができます。

### 手順

1. コンタクト一覧画面の上にある **検索バー** に、名前やメールアドレスの一部を入力します
2. **「検索」** ボタンを押します
3. 条件に合うお客さまだけが表示されます
4. 元に戻したいときは **「クリア」** ボタンを押します

### タグで絞り込む

「タグ」の選択欄から目印を選ぶと、そのタグがついたお客さまだけを表示できます。

> **「タグ」とは？**: お客さまにつける「付箋（ふせん）」のようなものです。たとえば「無料体験参加者」「購入済み」「VIP」など、自由に名前をつけて分類できます。

## 4.4　お客さまの詳しい情報を見る

### 手順

1. コンタクト一覧で、見たいお客さまの **名前** または **「詳細」** を押します
2. そのお客さまの詳細画面が表示されます

### 詳細画面で見られること

- **基本情報**: 名前、メールアドレス、電話番号、メモ
- **タグ**: つけられている目印（色付きラベル）
- **行動の記録**: いつ登録したか、どのフォームから来たか、メールを開いたかなどの履歴

## 4.5　お客さまの情報を編集する

### 手順

1. コンタクト詳細画面で **「編集」** ボタンを押します
2. 変更したい項目を書き換えます

| 入力欄 | 内容 |
|---|---|
| 名前 | お客さまの名前 |
| メールアドレス | 連絡先メール |
| 電話番号 | 電話番号（任意） |
| メモ | 自由にメモを書ける欄 |
| タグ | つけたい目印にチェックを入れる |

3. **「保存」** ボタンを押します

# 第5章　ファネル（集客ページ）を作る

## 5.1　ファネルとは

ファネルとは、**インターネット上に公開するページ（ホームページのようなもの）**のことです。

たとえば、次のようなページを簡単に作れます。

- 無料セミナーの申し込みページ
- 商品の紹介ページ
- お礼のページ（「お申し込みありがとうございます」と表示するページ）

このシステムでは、プログラミングの知識がなくても、**ブロック（部品）を並べるだけ**でページが作れます。積み木を組み合わせるイメージです。

## 5.2　新しいファネルを作る

### 手順

1. 左メニューの **「ファネル」** を押します
2. 右上の **「新規ファネル作成」** ボタンを押します
3. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| ファネル名 | このページセットの名前 | `無料セミナー申込` |
| スラッグ | 短い英語の名前 | `free-seminar` |

4. **「作成」** ボタンを押します

## 5.3　ページを追加する

1つのファネルの中に、複数のページを作れます。

### 手順

1. 作成したファネルの編集画面で、**「ページ追加」** ボタンを押します
2. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| ページタイトル | ページの名前 | `セミナー紹介ページ` |
| スラッグ | 短い英語の名前 | `intro` |
| ページの種類 | 用途を選ぶ | `ランディングページ` |

3. **「作成」** ボタンを押します

### ページの種類について

| 種類 | 使い道 |
|---|---|
| ランディングページ | 最初にお客さまが見るページ（チラシのようなもの） |
| サンクスページ | 申し込み後に表示する「ありがとう」ページ |
| セールスページ | 商品を詳しく紹介するページ |
| フォームページ | お名前やメールアドレスを入力してもらうページ |
| アップセルページ | 追加の商品をおすすめするページ |

## 5.4　ページにブロック（セクション）を追加する

ページの中身は「セクション」という**ブロック（部品）**を積み重ねて作ります。

### 手順

1. ページ編集画面の右側にある **「セクション追加」** ボタンを押します
2. 追加したいブロックの種類を選びます
3. ブロックが追加されます
4. ブロックの編集ボタン（鉛筆マーク）を押して、内容を書き換えます

### ブロック（セクション）の種類

| 種類 | できること | たとえると… |
|---|---|---|
| ヒーロー | 大きな見出しと説明文 | 看板・ポスターの見出し |
| テキスト | 自由な文章 | チラシの本文 |
| 画像 | 写真やイラストの表示 | チラシに貼る写真 |
| 動画 | YouTube等の動画を埋め込む | テレビCM |
| CTAボタン | 押せるボタン（申込ボタンなど） | 「ここを押してください」ボタン |
| フォーム | 名前やメールの入力欄 | 申込用紙 |
| カウントダウン | 締め切りまでの時間表示 | 残り時間タイマー |
| お客様の声 | 体験者のコメント表示 | 口コミ・レビュー |
| FAQ | よくある質問と答え | Q&Aコーナー |
| 区切り線 | ブロック間の仕切り | ページの区切り |

### ブロックの並び替え

ブロックの順番を変えたいときは、**ブロックをマウスで掴んで上下にドラッグ**（引きずる）します。好きな順番に並べ替えられます。

### ブロックの編集

1. 変更したいブロックの **鉛筆マーク**（編集ボタン）を押します
2. 内容を書き換えます
3. **「保存」** を押します

### ブロックの削除

1. 削除したいブロックの **ゴミ箱マーク**（削除ボタン）を押します
2. 確認メッセージが出たら **「OK」** を押します

## 5.5　フォームブロックの設定（重要）

フォームブロックは、お客さまに名前やメールアドレスを入力してもらうためのブロックです。これがあることで、お客さまの情報が自動的にコンタクト（顧客台帳）に登録されます。

### フォームの設定項目

| 項目 | 意味 | 例 |
|---|---|---|
| 送信ボタンの文字 | ボタンに表示する文字 | `無料で申し込む` |
| シナリオID | 申し込み後に自動メールを送る設定（後述） | `1` |
| リダイレクトURL | 申し込み後に表示するページのアドレス | `/p/free-seminar/thanks/` |

## 5.6　ページを公開する

作ったページをお客さまが見られるようにするには、「公開」の設定をします。

### 手順

1. ファネルの編集画面で **「公開する」** にチェックを入れます
2. **「更新」** ボタンを押します
3. 公開URLが表示されます（例: `/p/free-seminar/intro/`）
4. このURLにアクセスすると、作ったページが表示されます

> **ポイント**: 公開をやめたい場合は、チェックを外して「更新」を押せば、ページは非公開になります。

# 第6章　メール配信

## 6.1　メール配信でできること

メール配信機能は、お客さまに**メールを送る仕組み**です。2種類の送り方があります。

| 種類 | たとえると… | 使い道 |
|---|---|---|
| **シナリオ（ステップメール）** | 「入学おめでとう→1週間後に宿題→2週間後にテスト」のように決まった順番で届く手紙 | 申し込んでくれた人に、日にちをあけて順番に情報を届ける |
| **一斉配信（キャンペーン）** | 全員に同じチラシを配る | お知らせやセールの案内を全員に一度に送る |

## 6.2　シナリオ（ステップメール）を作る

シナリオとは、**「申し込んだ日から数えて、○日後にこのメールを送る」**という設定です。一度作っておけば、新しいお客さまが申し込むたびに自動でメールが届きます。

### シナリオを作る手順

1. 左メニューの **「メール配信」** を押します
2. **「シナリオ」** タブが選ばれていることを確認します
3. 右上の **「新規シナリオ作成」** ボタンを押します
4. シナリオ名を入力します（例: `無料セミナーフォローアップ`）
5. **「作成」** ボタンを押します

### ステップ（メールの1通1通）を追加する手順

1. 作成したシナリオの編集画面で **「ステップ追加」** ボタンを押します
2. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| ステップ番号 | 何番目のメールか | `1` |
| 遅延日数 | 申し込みから何日後に送るか | `0`（当日）|
| 遅延時間 | さらに何時間後か | `0` |
| 件名 | メールのタイトル | `お申込みありがとうございます` |
| 本文（テキスト） | メールの文面 | `田中様、この度は…` |
| 本文（HTML） | 装飾付きの文面（任意） | HTMLがわからなければ空でOK |

3. **「保存」** ボタンを押します
4. 2通目、3通目も同じように追加します

### ステップ設定の例

```
ステップ1: 0日後 → 「お申込みありがとうございます」
ステップ2: 1日後 → 「明日のセミナーについてのご案内」
ステップ3: 3日後 → 「セミナーはいかがでしたか？」
ステップ4: 7日後 → 「特別なご案内があります」
```

> **ポイント**: シナリオをファネルのフォームと連携させると、フォームに入力があった時点で自動的にメール配信が始まります（第5章「フォームブロックの設定」参照）。

## 6.3　一斉配信（キャンペーン）を作る

全員に同じメールを一度に送りたいときは、一斉配信を使います。

### 手順

1. メール配信画面で **「一斉配信」** タブを押します
2. 右上の **「新規一斉配信作成」** ボタンを押します
3. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| 配信名 | この配信の管理名 | `夏のセールお知らせ` |
| 件名 | メールのタイトル | `【期間限定】夏のセールのお知らせ` |
| 本文（テキスト） | メールの文面 | `いつもありがとうございます…` |
| 本文（HTML） | 装飾付きの文面（任意） | HTMLがわからなければ空でOK |
| 対象タグ | 特定のお客さまだけに送りたい場合に選択 | `購入済み` |

4. **「保存」** ボタンを押します

### メールを送信する

1. 保存した一斉配信の編集画面を開きます
2. 内容を確認します
3. **「送信」** ボタンを押します
4. 確認メッセージが出たら **「OK」** を押します

> **注意**: 送信したメールは取り消せません。送信前に必ず内容を確認しましょう。

### 配信ステータスの見かた

| 表示 | 意味 |
|---|---|
| 下書き（灰色） | まだ送信していない |
| 予約済み（黄色） | 送信予約がされている |
| 送信中（青） | 今まさに送信している |
| 送信済み（緑） | 送信が完了した |

# 第7章　商品と決済（お支払い）

## 7.1　商品管理でできること

この機能では、**売りたい商品を登録し、お客さまがインターネット上でお支払いできる**ようにします。

お支払いには「Stripe（ストライプ）」という安全な決済サービスを使います。クレジットカードでのお支払いに対応しています。

## 7.2　商品を登録する

### 手順

1. 左メニューの **「商品管理」** を押します
2. 右上の **「新規商品作成」** ボタンを押します
3. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|

# 第8章　会員サイト（学習コーナー）

## 8.1　会員サイトとは

会員サイトは、**商品を買ってくれたお客さまだけが見られる「特別な教室」**のようなものです。動画レッスン、テキスト教材、PDFファイルなどを届けることができます。

### 会員サイトの構造

```
会員サイト（教室の建物）
  └── コース（教科）
       └── レッスン（授業1回分）
```

たとえば…

```
田中料理教室 オンラインスクール（会員サイト）
  ├── 和食の基本コース（コース）
  │    ├── 第1回 だしの取り方（レッスン・動画）
  │    ├── 第2回 ご飯の炊き方（レッスン・動画）
  │    └── 第3回 味噌汁の作り方（レッスン・テキスト）
  └── お菓子作りコース（コース）
       ├── 第1回 クッキーの作り方（レッスン・動画）
       └── 第2回 ケーキの作り方（レッスン・PDF）
```

## 8.2　会員サイトを作る

### 手順

1. 左メニューの **「会員サイト」** を押します
2. **「新規サイト作成」** ボタンを押します
3. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| サイト名 | 会員サイトの名前 | `田中料理教室 オンライン` |
| スラッグ | 短い英語の名前 | `tanaka-cooking-online` |
| 説明 | サイトの紹介文 | `料理の基本を学べるオンラインスクールです` |
| 有効 | 公開するかどうか | チェックを入れる |

4. **「作成」** ボタンを押します

## 8.3　コース（教科）を作る

### 手順

1. 会員サイトの編集画面で **「コース追加」** ボタンを押します
2. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| コース名 | 教科の名前 | `和食の基本` |
| スラッグ | 短い英語の名前 | `washoku-basic` |
| 説明 | コースの紹介文 | `だし、ご飯、味噌汁を学びます` |
| サムネイル画像 | コースの表紙画像（任意） | 料理の写真をアップロード |
| 公開する | お客さまに見せるかどうか | チェックを入れる |

3. **「作成」** ボタンを押します

## 8.4　レッスン（授業）を作る

### 手順

1. コースの編集画面で **「レッスン追加」** ボタンを押します
2. 以下を入力します

| 入力欄 | 何を入れるか | 例 |
|---|---|---|
| レッスン名 | 授業のタイトル | `第1回 だしの取り方` |
| スラッグ | 短い英語の名前 | `lesson-1-dashi` |
| コンテンツの種類 | 動画・テキスト・PDFのどれか | `動画` |
| 本文 | テキストの内容 | レッスンの説明文 |
| 動画URL | YouTubeやVimeoのURL（動画の場合） | `https://youtube.com/watch?v=xxx` |
| ファイル | PDFファイル（PDFの場合） | ファイルをアップロード |
| プレビュー | 未購入でも見られるようにするか | チェックで「お試し公開」 |
| 公開する | お客さまに見せるかどうか | チェックを入れる |
| 並び順 | 表示の順番（数字が小さい方が先） | `1` |

3. **「作成」** ボタンを押します

### コンテンツの種類

| 種類 | 使い方 |
|---|---|
| 動画 | YouTube や Vimeo の動画URLを貼るだけで、画面に動画が表示されます |
| テキスト | 文章を書いて教材にできます |
| PDF | PDFファイルをアップロードして配布できます |

## 8.5　お客さまが会員サイトを使う流れ

商品を購入した後、お客さまはこのように会員サイトを使います。

### ログイン

1. お客さまが会員サイトのURL（例: `/m/tanaka-cooking-online/`）にアクセスします
2. 登録時のメールアドレスとパスワードを入力します
3. **「ログイン」** ボタンを押します
4. 受講できるコースの一覧が表示されます

### コースの受講

2. レッスンの一覧が表示されます
3. レッスンを1つ選んで内容を見ます
4. 学習が終わったら **「完了にする」** ボタンを押します
5. 進捗状況（何%終わったか）が自動で更新されます

### 進捗の表示

- 各コースに **進捗バー** が表示されます
- 例: 「3/10レッスン完了（30%）」
- 全レッスン完了すると、バーが緑色に変わります

### レッスン画面の操作

- **前のレッスンへ**: 画面下の左矢印ボタン
- **レッスン一覧に戻る**: 画面下の中央ボタン
- **次のレッスンへ**: 画面下の右矢印ボタン

# 第9章　ひととおりの流れをやってみよう

ここでは、システムの全機能を使って**「無料セミナーに申し込んでもらい、フォローメールを送り、有料講座を販売し、動画コンテンツを届ける」**という一連の流れを作ってみましょう。

## ステップ1: 会員サイトとコースを作る（教室を準備する）

1. 左メニュー → **「会員サイト」** → **「新規サイト作成」**
2. サイト名: `マイオンラインスクール`、スラッグ: `my-school`
3. **「コース追加」** → コース名: `動画講座・入門編`
4. レッスンを3つ追加:
   - 第1回「はじめに」（テキスト、プレビュー=オン）
   - 第2回「基礎知識」（動画）
   - 第3回「実践」（動画）
5. すべてのレッスンを「公開する」にチェック

## ステップ2: 商品を作る（売り物を準備する）

1. 左メニュー → **「商品管理」** → **「新規商品作成」**
2. 商品名: `動画講座・入門編`
3. 価格: `9800`（円）
4. 商品タイプ: `単品`
5. 購入後の自動処理:
   - コースアクセス権: `動画講座・入門編`（ステップ1で作ったコース）
   - タグを追加: `購入済み`
6. **「作成」** を押す

## ステップ3: メールのシナリオを作る（自動メールを準備する）

1. 左メニュー → **「メール配信」** → **「新規シナリオ作成」**
2. シナリオ名: `無料セミナーフォロー`
3. ステップを追加:

| ステップ | 遅延 | 件名 | 内容 |
|---|---|---|---|
| 1 | 0日後 | お申込みありがとうございます | セミナー申込のお礼 |
| 2 | 1日後 | セミナーの準備をしましょう | 予習のご案内 |
| 3 | 3日後 | セミナーはいかがでしたか？ | 感想のお願い |
| 4 | 5日後 | 特別なご案内 | 有料講座の紹介 |

## ステップ4: ファネル（集客ページ）を作る（チラシを作る）

1. 左メニュー → **「ファネル」** → **「新規ファネル作成」**
2. ファネル名: `無料セミナー`、スラッグ: `free-seminar`

### ページ1: 申し込みページ

1. **「ページ追加」** → タイトル: `セミナー紹介`、種類: `ランディングページ`
2. 以下のセクションを追加:
   - **ヒーロー**: 見出し「無料オンラインセミナー開催！」
   - **テキスト**: セミナーの紹介文
   - **お客様の声**: 過去の参加者の感想
   - **フォーム**: メールアドレス入力欄
     - シナリオID: （ステップ3で作ったシナリオのID）
     - リダイレクトURL: `/p/free-seminar/thanks/`

### ページ2: お礼ページ

1. **「ページ追加」** → タイトル: `お申込み完了`、種類: `サンクスページ`、スラッグ: `thanks`
2. 以下のセクションを追加:
   - **ヒーロー**: 見出し「お申込みありがとうございます！」
   - **テキスト**: 次のステップの案内

### 公開する

1. ファネル編集画面で **「公開する」** にチェック → **「更新」**

## ステップ5: 動作を確認する

1. 公開URL（例: `/p/free-seminar/intro/`）にアクセス
2. セミナー紹介ページが表示される
3. フォームにメールアドレスを入力 → 送信
4. サンクスページが表示される
5. コンタクト一覧に新しいお客さまが登録されている
6. シナリオのメールが自動で送られ始める（コンソールに表示）

## 完成した仕組みのイメージ

```
お客さまが紹介ページを見る
  ↓
フォームにメール入力して送信
  ↓
コンタクト（顧客台帳）に自動登録
  ↓
シナリオメールが自動スタート
  ↓  ↓  ↓  ↓
  0日後: お礼メール
  1日後: 準備のご案内
  3日後: 感想のお願い
  5日後: 有料講座の紹介メール ← ここに購入リンクを入れる
  ↓
お客さまが有料講座を購入（Stripe決済）
  ↓
自動でコースのアクセス権が付与される
  ↓
お客さまが会員サイトで動画レッスンを受講できる
```

# 第10章　困ったときは（よくある質問）

## Q1. ログインできません

**A.** 以下を確認してください。
- ユーザー名とパスワードが正しいか（大文字・小文字の違いに注意）
- Caps Lock（大文字固定）がオンになっていないか
- 正しいURLにアクセスしているか（ http://127.0.0.1:8000/login/ ）

## Q2. プロジェクトが表示されません

**A.** プロジェクトをまだ作成していない可能性があります。ログイン後、プロジェクト一覧画面で「新規作成」ボタンを押してプロジェクトを作成してください。

## Q3. ファネルページが見られません

**A.** 以下を確認してください。
- ファネルが「公開」になっているか（編集画面で「公開する」にチェックが入っているか）
- URLが正しいか（ `/p/ファネルのスラッグ/ページのスラッグ/` ）

## Q4. メールが届きません

**A.** 開発環境では、メールは実際には送信されず、コンソール（黒い画面）に表示されます。本番環境で実際にメールを送るには、メールサーバーの設定が必要です。

## Q5. 決済のテストはどうすればいいですか？

**A.** Stripeの「テストモード」を使います。テスト用のクレジットカード番号は `4242 4242 4242 4242` です（有効期限は未来の日付、CVCは任意の3桁）。実際のお金は引き落とされません。

## Q6. 画像がアップロードできません

**A.** 以下を確認してください。
- ファイルの形式がJPEG、PNG、GIFのいずれかであること
- ファイルのサイズが大きすぎないこと（目安: 5MB以下）

## Q7. セクションの順番を変えたいです

**A.** ページ編集画面で、セクション（ブロック）をマウスで掴んで上下にドラッグしてください。好きな順番に並べ替えられます。

## Q8. コンタクト（お客さま）を削除したいです

**A.** コンタクトの詳細画面から削除できます。ただし、削除すると関連する注文履歴やメール配信の記録もなくなりますので、慎重に行ってください。

## Q9. 画面の左メニューが見えません

**A.** ブラウザの画面を広げてみてください。画面が狭いとメニューが隠れることがあります。

## Q10. 操作を間違えてデータを消してしまいました

**A.** 削除したデータは元に戻せません。大切なデータは、操作前に念のため情報をメモしておくことをおすすめします。
